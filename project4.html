Классификатор электротехнических товаров по товарной номенклатуре внешнеэкономической деятельности (ТНВЭД)

Классификатор позволяет определить группу товара по короткому описанию с точностью 95%. При этом всего используется 411 групп товаров.

  • Для решения задачи классификации электротехнических товаров по ТНВЭД была обучена модель XGBoost.
  • Всего для обучения использован размеченный набор данных, содержащий около 70 тысяч строк описаний товаров, заполненных вручную специалистами профильного отдела.
  • Общее количество классов 411.
  • Точность классификации, оцененная на тестовом наборе, составила 95%.
  • Для применения модели был разработан Телеграм бот, который доступен в круглосуточном режиме при запуске скрипта на локальном компьютере.
