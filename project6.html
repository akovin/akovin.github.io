Интеллектуальная система оптимизации застройки земельных участков

Интеллектуальная система для проектировщиков, которая автоматически подбирает оптимальные параметры застройки на основе машинного обучения. Основные этапы: Загрузка DXF-файла с границами участка Рассчет возможных вариантов размещения зданий (этажность, тип, отступы) Использует CatBoost для предсказания максимальной жилой площади Визуализирует топ-10 вариантов с детальными характеристиками Дополнительно включает калькулятор ТЭП для расчета социальной инфраструктуры Результат использования системы: сокращение времени на проектирование и повышение эффективности использования площади под застройку.
